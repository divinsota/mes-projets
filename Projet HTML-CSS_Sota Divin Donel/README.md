# Projet Site E-commerce - Association

## Description
Ce projet consiste en la création d'un site web de vente en ligne pour une association (WWF, Handicap International, SPA, etc.).  
Il s'agit d'un site complet développé en **HTML et CSS**, respectant à la fois la qualité du code et le rendu visuel.  

Le site inclut :
- Page d'accueil avec hero section, produits et catégories mis en avant
- Pages catégories (au moins 4)
- Pages descriptives pour chaque produit (au moins 8)
- Page support / panier avec formulaire fonctionnel
- Page d'erreur 404

---

## Technologies utilisées
- HTML5
- CSS3
- Responsive design pour Desktop, Mobile et Tablette

---

## Fonctionnalités principales

### 1. Page d'accueil
- Header avec barre de navigation et logo centré
- Menu simple et cohérent
- Hero section avec texte
- Mise en avant de produits et catégories
- Footer avec menu, icônes de réseaux sociaux et informations légales

### 2. Pages catégories
- Grille de prévisualisation des produits avec image, nom et prix
- Redirection vers les pages descriptives
- Cohérence avec le design général du site

### 3. Pages descriptives
- Titre et description complète du produit
- Au moins 2 images par produit
- Caractéristiques détaillées
- Sélecteur de quantité et bouton "Ajouter au panier"
- Suggestions de produits similaires

### 4. Page Support / Panier
- Formulaire de contact fonctionnel
- Réception des messages sur boîte mail

### 5. Page 404
- Affichage d’un message d’erreur
- Bouton de redirection vers l’accueil

---

## Responsive & Accessibilité
- Site responsive pour **Desktop, Mobile et Tablette**
- Respect des critères d’accessibilité selon le RGAA et WCAG
- Aucun lien mort sur le site

---

## Design et optimisation
- Cohérence graphique et esthétique générale
- Hiérarchie claire des contenus
- Effets visuels simples et lisibles
- Code HTML et CSS structuré, indentation correcte et classes cohérentes
- Arborescence claire des fichiers

---

## Comment l’utiliser
1. Télécharger ou cloner ce repository
2. Ouvrir le fichier `index.html` dans un navigateur web
3. Naviguer sur le site pour explorer toutes les pages et fonctionnalités
